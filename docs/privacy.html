Privacy Policy

Last updated: February 17, 2026

Overview

IRIS Migration Checklist is a browser extension that exports server configuration from InterSystems IRIS/Caché servers and compares snapshots between two servers. This privacy policy explains what data the extension accesses, how it is stored, and how it is used.

What data is accessed

When you click "Save Server Data" or "Compare to Saved Server", the extension fetches a configuration export from the IRIS/Caché server you are currently authenticated to. This export may include:

  • Namespace configuration (names, database mappings)
  • Scheduled tasks (names, schedules, class references)
  • Web application definitions (paths, dispatch classes, authentication settings)
  • SQL gateway connections (driver, URL — passwords are not included)
  • User accounts (usernames, roles — passwords are never exported)
  • Security roles and resources (names, descriptions)
  • Per-namespace metadata: class names, global names, credential names (not passwords), production items, and lookup table entries

How data is stored

All data is stored locally in browser.storage.local, or chrome.storage.local, which is sandboxed to this extension on your machine. No data is sent to any external server, cloud service, or third party. The only network communication happens directly between your browser and the IRIS/Caché server you choose to connect to.

The extension stores two items locally:

  • Saved snapshot — the most recent server configuration export you saved.
  • Comparison report — the most recent diff report generated by comparing two snapshots.

You can delete all stored data at any time using the "Clear Saved Data" button in the extension popup. Uninstalling the extension also removes all stored data.

How data is used

Data is used exclusively to provide the extension's core functionality: saving a configuration snapshot and comparing it against another server's snapshot. Data is never used for advertising, analytics, profiling, or any purpose other than generating the migration comparison report.

Data sharing

This extension does not share, sell, or transfer user data to any third party. No data leaves your browser except for the authenticated requests made directly to the IRIS/Caché server you are connected to.

Backend installation disclosure

If the export API does not exist on the target server, the extension can install a backend class (Migration.Framework.cls) via the Atelier REST API. This action:

  • Uploads and compiles a single ObjectScript class in the %SYS namespace.
  • Registers a web application at /api/v1/migration/framework.

This installation only happens with your explicit consent — the extension asks for confirmation before making any changes to the server.

Permissions

The extension requests the following browser permissions:

  • storage — to persist snapshots and reports locally.
  • activeTab — to read the URL of the current tab and determine the server origin.
  • Host permissions for /api/v1/migration/* and /api/atelier/* — to communicate with the IRIS/Caché server APIs. These are scoped to the specific API paths the extension needs.

Limited Use disclosure

The use of information received from browser APIs adheres to the Chrome Web Store User Data Policy and Firefox Add-on Policies, including the Limited Use requirements. Data obtained through this extension is used solely for the purpose of generating migration comparison reports and is not transferred to third parties, used for advertising, or used to determine creditworthiness.
